# BVOX LAB v1

A Vite + GSAP single-page landing experience for BVOX LAB.

## Stack
- Vite
- Vanilla JS
- GSAP

## Local Dev
- Install: `npm install`
- Dev server: `npm run dev`
- Build: `npm run build`
- Preview build: `npm run preview`

## Deploy
- Platform: Vercel
- Build command: `npm run build`
- Output directory: `dist`
